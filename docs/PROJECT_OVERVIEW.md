# Project Overview

## What Is This?

A standalone interactive audit presentation website for **SPARK Schools**, built by **Broadbrand** (a digital agency within the DSG group). The site presents a concise mini-audit covering SEO, PPC, and broader marketing context as part of a pitch selection process.

## Target Audience

SPARK Schools decision-makers evaluating Broadbrand's capabilities and strategic recommendations.

## Deployment

- **Platform:** Vercel
- **URL:** TBD (spark-audit.vercel.app or similar)
- **Framework:** Next.js 16+ (App Router) with static export

## Companion Site

This audit site is designed to complement the RFQ proposal site deployed at [sparkrfq.vercel.app](https://sparkrfq.vercel.app/). Both share the same brand CI for continuity.

## Owner

Broadbrand Digital Agency / DSG Group

## Tech Stack

- Next.js 16.1.6 (App Router, TypeScript)
- Tailwind CSS v4.2
- shadcn UI (base-nova style, @base-ui/react)
- Lucide React icons
- Google Fonts (Sofia Sans + Poppins)
