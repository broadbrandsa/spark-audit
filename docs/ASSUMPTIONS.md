# Assumptions

## Data Sources

All audit data is based on **January–December 2025** exports from:

- **Google Search Console (GSC)** — organic search performance, impressions, clicks, CTR
- **Google Analytics 4 (GA4)** — traffic, conversions, funnel data, channel attribution
- **Google Ads** — campaign spend, clicks, conversions, cost metrics
- **Meta Ads (Facebook/Instagram)** — campaign performance, lead costs, impressions
- **PageSpeed Insights** — Core Web Vitals, performance scores, TTFB

## Directional Nature

This is a **directional audit**, not a forensic analysis. Findings are based on available data exports and surface-level technical checks. Full validation requires:

- Direct access to GSC, GA4, and ad platform accounts
- Server-side log analysis
- CRM data integration for true cost-per-acquisition figures

## Key Assumptions

1. Data exports are representative of the full reporting period
2. Conversion tracking is correctly configured across platforms
3. Attribution models in GA4 reflect actual user journeys
4. Meta Ads cost figures include all campaign types (not just lead gen)
5. The 84/16 Meta/Google budget split is intentional, not accidental
