# Scope

## Site Structure

Single-page website with 7 content sections plus navigation and footer:

1. **Hero** — Full-screen intro with key audit stats
2. **Executive Summary** — Score cards, top 5 opportunities, quick wins
3. **SEO Audit** — Score breakdown, tabbed deep-dive (Crawlability, Performance, On-Page, Content, Keywords)
4. **PPC Audit** — Google Ads + Meta Ads analysis, budget allocation, campaign performance
5. **Marketing Context** — Channel performance matrix, funnel breakdown, emerging signals
6. **Recommendations** — 30 prioritized actions across 4 tiers (Critical → Strategic)
7. **Proposal Bridge** — Connects audit findings to strategic themes from the RFQ proposal

## Companion Site

This site works alongside the RFQ proposal at [sparkrfq.vercel.app](https://sparkrfq.vercel.app/). The Proposal Bridge section explicitly links audit data back to proposal strategy.

## What This Is NOT

- No backend / API
- No authentication
- No CMS — all content is hardcoded in components
- No database
- No user-generated content
- Not a full technical SEO audit tool
